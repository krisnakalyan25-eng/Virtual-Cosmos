# 🌌 Proximity Chat App (Realtime)

A real-time interactive web application where users can move in a virtual space and chat with nearby users.

---

## 🚀 Features

* 🧑 User enters name and joins world
* 🟦 Move using arrow keys
* 🟥 See other users in real-time
* 📍 Proximity-based chat (chat appears only when near)
* 💬 Real-time messaging using WebSockets
* ✍️ Typing indicator
* 🖼️ User avatars
* 👥 Active users count
* 🗄️ MongoDB user storage

---

## 🛠️ Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Node.js + Express
* Realtime: Socket.io
* Database: MongoDB

---

## ⚙️ Setup Instructions

### 1. Clone repo

```
git clone <your-repo-link>
cd project
```

### 2. Backend setup

```
cd server
npm install
npm start
```

### 3. Frontend setup

```
cd client
npm install
npm run dev
```

---

## 🎮 How it Works

1. Enter your name
2. Move using arrow keys
3. Approach another user
4. Chat box appears automatically
5. Move away → chat disappears
